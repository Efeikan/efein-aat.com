# Efe İnşaat Web Sitesi

Bu proje [Next.js](https://nextjs.org) ile oluşturulmuştur.

## Geliştirme Sunucusunu Başlatma

```bash
npm run dev
```

Tarayıcında [http://localhost:3000](http://localhost:3000) adresini aç.
